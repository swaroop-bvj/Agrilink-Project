====== AGRILINK INSTALLATION =====

STEP-1: Install XAMPP SERVER from online

STEP-2: Copy the agrilink_api(folder) and paste it on C:\Xampp\htdocs\

STEP-3: Open Xampp Server
* Click on MySQL & Apache start button
* Click on MySQL Admin 
* Click on SQL
* Paste the SQL Query(which is on the Agrilink folder)
* Click on go

STEP-4: Open node_modules
* open Agrilink.bat(edit in notepad)
* replace your file path

STEP-5: Click on Agrilink Shortcut
* Make registration
* Login and use the application
