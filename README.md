# AgriLink 🌾
> "It's not just an Idea, It's a life of a Farmer"

## Overview
AgriLink is a comprehensive full-stack web and mobile application designed to modernize agricultural practices and empower farmers. By leveraging Machine Learning and robust web technologies, AgriLink provides an end-to-end digital ecosystem that helps farmers make data-driven decisions, connect with local markets, and access real-time support. 

## Key Problem Solving
This platform was specifically engineered to resolve five fundamental challenges faced by the farming community:
1. **Intelligent Crop Suggestions:** Utilizes predictive modeling to recommend the most profitable and viable crops to plant.
2. **Cultivation Guidance:** Provides step-by-step, actionable advice on *how* to grow and maintain specific crops effectively.
3. **Direct-to-Market Sales:** A transparent platform where farmers can list and sell their harvested crops directly to buyers.
4. **Agricultural Supply Marketplace:** A dedicated e-commerce hub for purchasing trusted seeds, fertilizers, and farming equipment.
5. **Farmer Support Network:** A reliable helpdesk connecting farmers with experts and resources for troubleshooting and agricultural advice.

## Tech Stack
* **Frontend:** React Native, JavaScript, HTML
* **Backend:** PHP (RESTful APIs)
* **Database:** MySQL (hosted via XAMPP Server)
* **Machine Learning:** Python, XGBoost

## Project Architecture
AgriLink integrates a mobile-first frontend with a lightweight PHP backend, ensuring fast and reliable API communication. The integrated Machine Learning models process agricultural data to generate real-time insights, all stored securely in a relational MySQL database.
